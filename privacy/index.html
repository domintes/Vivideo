Vivideo Privacy Policy

Vivideo is a browser extension that provides real-time video enhancement. This Privacy Policy explains how Vivideo handles user data.

1. Data Collection

Vivideo does not collect, transmit, or store any personally identifiable information (PII). All enhancement settings and profiles are stored locally in your browser and remain on your device.

2. Permissions

The extension requires certain permissions to function properly:

  • activeTab: To apply video enhancements on the currently active page.
  • storage: To save your custom video enhancement settings locally.
  • scripting: To inject scripts that apply the enhancements to videos.
  • downloads: Optional for any export functionality you may use locally.
  • host permissions: To target video elements on web pages. No data is sent to external servers.

3. Remote Code

Vivideo does not use any remote code. All scripts are included in the extension package.

4. Third-Party Sharing

No user data is shared with third parties. Vivideo operates entirely locally on your device.

5. Compliance

By using Vivideo, you acknowledge that all settings and enhancements are applied locally and no personal information leaves your device. This extension complies with Chrome Web Store policies regarding user data.
